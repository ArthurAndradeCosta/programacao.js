Resumão

CSS (Cascading Style Sheets) é uma linguagem usada para definir o estilo e o layout de páginas HTML, controlando cores, fontes, tamanhos, espaçamentos, bordas e posicionamentos. Ele separa o conteúdo da aparência, tornando o código mais organizado e fácil de manter. Pode ser aplicado de forma inline, interna ou externa, sendo o arquivo externo o método mais comum. O CSS permite criar sites visualmente atraentes, responsivos e compatíveis com diferentes dispositivos, além de seguir princípios de cascata, herança e especificidade para determinar como os estilos são aplicados.

Seletores

Os seletores em CSS determinam quais elementos do HTML serão estilizados. Eles permitem direcionar regras para tags, classes, identificadores ou grupos de elementos específicos, tornando o controle visual da página mais preciso e organizado. A escolha do seletor adequado influencia diretamente a eficiência e a clareza do código.

Propriedades e valores

As propriedades definem o tipo de estilo a ser aplicado, enquanto os valores determinam como esse estilo será exibido. Cada elemento pode ter múltiplas propriedades configuradas, como aparência, tamanho, cor, espaçamento e comportamento visual, proporcionando flexibilidade total no design da página.

Cascata e especificidade

A cascata estabelece a ordem em que as regras de estilo são aplicadas, considerando a hierarquia e a origem de cada uma. Já a especificidade define qual regra tem prioridade quando há conflitos. Essa combinação garante previsibilidade e controle sobre o resultado visual final.

Box Model

O modelo de caixa é a base do layout em CSS. Cada elemento é tratado como uma caixa composta por conteúdo, espaçamento interno, borda e margem. Entender essa estrutura é essencial para organizar o espaço, alinhar elementos e ajustar proporções na página.

Design responsivo

O design responsivo adapta o layout da página a diferentes tamanhos de tela, garantindo boa aparência e funcionalidade em computadores, tablets e celulares. Ele é fundamental para a acessibilidade e a experiência do usuário em dispositivos variados.

Layout e posicionamento

CSS oferece diversas formas de organizar os elementos na página, controlando posições, alinhamentos e distribuições. Ferramentas como o posicionamento absoluto, flexível e em grade permitem criar estruturas visuais complexas e dinâmicas com precisão.
