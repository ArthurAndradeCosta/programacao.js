Básico do css

Eu aprendi apenas o básico, e eles está aqui: a tag link "stylesheet" href="style.css criou uma página apenas para eu controlaro css, onde não é possível usar qualquer tipo de tag, como !. Para editar lá, você escreve body, e todo estilo lá será adicionado para o site todo, lá também se usa {} como método para editar, exemplo de código:
h1 { color: rgb(252, 252, 253); font-family: 'Segoe UI', Tahoma, Geneva, Verdana, sans-serif; font-size: 22px; }
