Parágrafos

A escrita foi a primeira coisa que eu aprendi, com parágrafos, títulos e até algumas esccritas diferentes, assim que se usa a tag ! vc deve coloca-la em pt-br e adicionar um título a essa página, esse título aparecerá na topo da janela. A tag h1 cria o primeiro real título onde a escrita é maior, em h2 ela cria o segundo título que normalmente é menor, e assim consecutivamente até o h6, além da tag hr que se cria uma barra abaixo do título, logo em seguida se usa a tag "p" que é o parágrafo e onde isso está sendo escrito. Normalmente independente de como se escreve no vscode ele trasmite o texto totalmente alinhado, por isso seusa a tag br que funciona para a quebra de linhas.

Símbolos e versões de escritas

existem alguns símbolos que é normal usar em sites, como ® ™ ou ™, para usar esses símbolos você usa a tag & (e o nome do símbolo) como & trade;

Escrita

Se deve usar tag em toda a escrita de um site, como usar negrito ou itálico que se usam a tag b e i para isso, também a tag mark e style para marcar uma palavra, outras tags como del para fazer este risco, ou até números elevados ou abaixo com a tag sup e sub 10 10, também existe a tag code que deixa as letras de mesmo espaçamento ou até a tag "pre" que serve para deixar a escrita do jeito que você deixou.
